# Edge Overlays - Issue Tracker

This repository is the **public issue tracker** for Edge Overlays.

## Report a Bug

Found a bug? [Open an issue](../../issues/new?template=bug_report.md) and we'll look into it.

## Request a Feature

Have an idea? [Submit a feature request](../../issues/new?template=feature_request.md).

## Guidelines

- **Search first** - Check if your issue already exists
- **One issue per report** - Keep issues focused
- **Be specific** - Include steps to reproduce, expected vs actual behavior
- **Include context** - OS, browser, version info helps us debug faster
